Introduction

  1. What is Machine Learning?

    Machine learning is the ability to allow a software system to "learn" through data-driven predictive programming. Through this technique, the software creates a model using data analysis from data that is provided to it. The model created by the software can then be used to solve complex data-intensive problems such as:

    • Pattern recognition
    • Regression analysis
    The software "learns" by iterating on data continuously to improve the data-model. Machine learning gives software the ability to evolve and adapt to new data. At the philisophical level, machine learning is similar to how humans learn. In a sense, machine learning techniques enable software to learn complex relationships similarly to how humans learn; although, probably at a much more simplified level.

  2. Why use Machine Learning?

    Machine learning has many applications, especially of use in real-world applications. The goal of machine learning is to extract useful information from complex datasets. Thus, effective application of machine learning techniques can solve complex and dynamic problems. Furthermore, these solutions can evolve and adapt to the problem over time as long as new data is provided for the software to "learn".

    Some example applications of machine learning include:

    • Computer vision (object recognition)
    • Email spam detection
    • Natural language processing
    • General game playing (artificial intelligence)

  3. Implementing Machine Learning

    There are three main types of learning a software system can undergo:

    • Reinforcement learning
    • Unsupervised learning
    • Supervised learning
